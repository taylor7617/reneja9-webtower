# RENÉJA9 WebTower · Digital Bone Collector

This is the RENÉJA9 WebTower — a spooky, abandoned-spacecraft-style control room
that turns three tools into a single **Web Skeleton Factory**:

- Google (GI login + Drive)
- GitHub (repos)
- ChatGPT / Gemini (skeleton generation)

## Core Mission

1. **GI Login**
   - Use one Gmail account for:
     - Gmail
     - Google Drive
     - GitHub account
     - ChatGPT / Gemini
   - GI = Digital Access, Verification, Identification, Domain.

2. **Generate Web Skeleton**
   - In ChatGPT / Gemini, request:
     - “Generate a one-page static website skeleton for: [PROJECT NAME]”
   - Copy the generated `index.html` into a new GitHub repo.

3. **Publish as WebTower**
   - Enable GitHub Pages on the repo.
   - The site becomes a **Tower** in the Web grid.

## Usage Pattern

1. Pick a new client or idea.
2. Use the GI login to open:
   - Gmail
   - Drive
   - GitHub
   - ChatGPT / Gemini
3. Ask Gemini/ChatGPT to generate the **web skeleton**.
4. Create repo → paste code → publish.
5. Add the new Tower link to:
   - FlightLine maps
   - Memphis DC grids
   - Alumni / business universcity.
